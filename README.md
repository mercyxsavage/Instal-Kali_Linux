Script Untuk Menginstal Kali-linux Pada Termux.

Bagaimana cara menggunakan ?

-) git clone https://github.com/JCyber28/Instal-Kali-Linux
-) cd Instal-Kali-Linux
-) cp Kali-linux $ HOME
-) cd $ HOME
-) chmod + x Kali-linux
-) ./ Kali-linux
